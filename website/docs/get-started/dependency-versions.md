# Dependency versions

::: info TL;DR

- Module Support: ECMAScript Modules (ESM) only
- ESLint: `^9.15.0` (Flat Config)
- Node.js: `^20.19.0 || ^22.13.0 || >=24.0.0`

:::

## Module Support

## ESLint

The version range of ESLint currently supported is [`^9.15.0`](https://github.com/eslint/eslint/releases/tag/v9.15.0).

## Node.js

In line with the [ESLint project's prerequisites](https://github.com/eslint/eslint?tab=readme-ov-file#installation-and-usage), we support Node.js versions `^20.19.0 || ^22.13.0 || >=24.0.0`.
